연습 문제 9.2.1의 답¶

로그가능도함수는 다음과 같다.

\[\begin{split} \begin{align} \begin{aligned} & \log L(\mu; x_1=1, x_2=0, x_3=1, x_4=1) \\ &= \log p(x_1=1, x_2=0, x_3=1, x_4=1; \mu) \\ &= \log \big( p(x=1;\mu)p(x=0;\mu)p(x=1;\mu)p(x=1;\mu) \big) \\ &= \log \big( \mu \cdot (1-\mu) \cdot \mu \cdot \mu \big) \\ &= \log \big( \mu^3(1-\mu)\big) \end{aligned} \tag{A.9.1} \end{align} \end{split}\]

이 로그가능도함수를 최대화하는 모수의 값을 찾기위해 미분한 도함수가 0이 되는 위치를 찾는다.

\[ \begin{align} \dfrac{d\log L}{d\mu} = \frac{3\mu^2 - 4\mu^3}{\mu^3(1-\mu)} = 0 \tag{A.9.2} \end{align} \]
\[ \begin{align} \mu = \frac{3}{4} \tag{A.9.3} \end{align} \]

연습 문제 9.2.2의 답¶

\(K=4\)인 카테고리분포의 확률질량함수는 다음과 같다.

\[ \begin{align} P(x_; \mu) = \mu_1^{x_1} \mu_2^{x_2} \mu_3^{x_3} \mu_4^{x_4} \tag{A.9.4} \end{align} \]

따라서 전체 데이터에 대한 로그가능도함수는

\[\begin{split} \begin{align} \begin{aligned} & \log L(\mu; x_1=1, x_2=4, x_3=1, x_4=2, x_5=4, x_6=2, x_7=3, x_8=4) \\ &= \log p(x_1=1, x_2=4, x_3=1, x_4=2, x_5=4, x_6=2, x_7=3, x_8=4; \mu) \\ &= \log \big( p(x=1;\mu)p(x=4;\mu)p(x=1;\mu)p(x=2;\mu)p(x=4;\mu)p(x=2;\mu)p(x=3;\mu)p(x=4;\mu) \big) \\ &= \log \mu_1 \mu_4 \mu_1 \mu_2 \mu_4 \mu_2 \mu_3 \mu_4 \\ &= \log \mu_1^2 \mu_2^2 \mu_3 \mu_4^3 \\ &= 2\log \mu_1 + 2\log \mu_2 + \log \mu_3 + 3\log \mu_4 \end{aligned} \tag{A.9.5} \end{align} \end{split}\]

그런데 이 모수값은 다음 제한조건을 만족해야 한다.

\[ \begin{align} \mu_1 + \mu_2 + \mu_3 + \mu_4 - 1 = 0 \tag{A.9.6} \end{align} \]

이 제한조건을 만족하면서 로그가능도함수를 최대화하는 모수의 값을 찾기위해 라그랑주방법을 사용하여 새로운 목적함수 \(J\)를 만든다.

\[ \begin{align} J = 2\log \mu_1 + 2\log \mu_2 + \log \mu_3 + 3\log \mu_4 + \lambda(\mu_1 + \mu_2 + \mu_3 + \mu_4 - 1) \tag{A.9.7} \end{align} \]

목적함수 \(J\)를 모수와 라그랑주승수로 미분하여 그레디언트가 0이 되는 값을 찾는다.

\[\begin{split} \begin{align} \begin{aligned} \dfrac{\partial J}{\partial \mu_1} &= \dfrac{2}{\mu_1} + \lambda = 0 \\ \dfrac{\partial J}{\partial \mu_2} &= \dfrac{2}{\mu_2} + \lambda = 0 \\ \dfrac{\partial J}{\partial \mu_3} &= \dfrac{1}{\mu_3} + \lambda = 0 \\ \dfrac{\partial J}{\partial \mu_4} &= \dfrac{3}{\mu_4} + \lambda = 0 \\ \dfrac{\partial J}{\partial \lambda} &= \mu_1 + \mu_2 + \mu_3 + \mu_4 - 1 = 0 \\ \end{aligned} \tag{A.9.8} \end{align} \end{split}\]

이 연립방정식을 풀면

\[ \begin{align} \mu_1 = \frac14, \;\;\mu_2 = \frac14, \;\;\mu_3 = \frac18, \;\;\mu_4 = \frac38 \tag{A.9.9} \end{align} \]

연습 문제 9.3.1의 답¶

사전분포가 하이퍼모수 \(a=b=1\)인 베타분포인 경우 사후분포는 다음과 같은 하이퍼모수를 가지는 베타분포가 된다.

상품 A의 사후분포의 하이퍼모수:

\[\begin{split} \begin{align} \begin{aligned} a'_A &= 2 + 1 = 3 \\ b'_A &= 1 + 1 = 2 \end{aligned} \tag{A.9.10} \end{align} \end{split}\]

상품 B의 사후분포의 하이퍼모수:

\[\begin{split} \begin{align} \begin{aligned} a'_B &= 60 + 1 = 61 \\ b'_B &= 40 + 1 = 41 \end{aligned} \tag{A.9.11} \end{align} \end{split}\]
aA, bA = 3, 2
aB, bB = 61, 41
xx = np.linspace(0, 1, 1000)
plt.plot(xx, sp.stats.beta(aA, bA).pdf(xx), 
         c="g", ls="-", label="상품 A의 모수")
plt.plot(xx, sp.stats.beta(aB, bB).pdf(xx), 
         c="r", ls="--", label="상품 B의 모수")
plt.legend()
plt.title("베이즈 추정법으로 계산한 베르누이분포 모수의 사후분포")
plt.show()

연습 문제 9.3.2의 답¶

사이킷런에서 제공하는 붓꽃데이터 중 꽃받침의 길이를 정규분포라고 가정하고 베이즈 추정법을 사용하여 기댓값을 추정하라. 단, 정규분포의 분산으로는 표본분산을 사용하고

사전분포가 기댓값 1, 분산 1인 정규분포이므로

\[\begin{split} \begin{align} \begin{aligned} \mu_0 &= 1 \\ \sigma_0^2 &= 1 \end{aligned} \tag{A.9.12} \end{align} \end{split}\]

사이킷런에서 제공하는 붓꽃데이터 중 꽃받침의 길이의 표본평균과 표본분산을 구하면 5.84와 0.68이다.

from sklearn.datasets import  load_iris
iris = load_iris()
x = iris.data[:, 0]
N = len(x)
mu = np.mean(x)
sigma2 = np.var(x)
print("sample mean:", mu)
print("sample variance:", sigma2)
sns.distplot(x, kde=False, fit=sp.stats.norm)
plt.vlines(x=mu, ymin=0, ymax=0.6, ls="--")
plt.vlines(x=mu+2*np.sqrt(sigma2), 
           ymin=0, ymax=0.6, ls=":")
plt.vlines(x=mu-2*np.sqrt(sigma2), 
           ymin=0, ymax=0.6, ls=":")
plt.show()

sample mean: 5.843333333333334
sample variance: 0.6811222222222223

베이즈 추정법을 사용하여 사후분포를 구하면 다음과 같다.

\[\begin{split} \begin{align} \begin{aligned} \mu'_0 &= \dfrac{\sigma^2}{N\sigma_0^2 + \sigma^2}\mu_0 + \dfrac{N\sigma_0^2}{N\sigma_0^2 + \sigma^2} \dfrac{\sum x_i}{N} \approx 5.82 \\ \sigma_0^{'2} &= \left( \dfrac{1}{\sigma_0^{2}} + \dfrac{N}{\sigma^{2}} \right)^{-1} \approx 0.0045 \end{aligned} \tag{A.9.13} \end{align} \end{split}\]
mu0 = 1
sigma02 = 1

mu_post = sigma2 / (N * sigma02 + sigma2) * mu0 \
          + (N * sigma02) / (N * sigma02 + sigma2) * mu
sigma2_post = 1 / (1 / sigma02 + N / sigma2)
print("post meam:", mu_post)
print("post variance:", sigma2_post)

post meam: 5.821440066849044
post variance: 0.00452028901946797

plt.subplot(211)
sns.distplot(x, kde=False, fit=sp.stats.norm)
plt.vlines(x=mu_post, ymin=0, ymax=0.6, ls="--")
plt.xlim(4, 8)
plt.subplot(212)
xx = np.linspace(4, 8, 1000)
plt.plot(xx, 
         sp.stats.norm(loc=mu_post, 
                       scale=np.sqrt(sigma2_post)).pdf(xx))
plt.title("기댓값의 사후분포")
plt.xlim(4, 8)
plt.tight_layout()
plt.show()

                            alternative="greater"))

연습 문제 9.5.5의 답¶

두 카테고리 분포의 모수를 비교검정하려면 chi2_contingency()를 사용한다. 모수가 같은 것이 디폴트 귀무가설이므로 다음과 같이 유의확률을 계산할 수 있다. 유의확률이 0.7%이므로 두 분포의 모수가 다르다. 즉, 데이터 사이언스 스쿨 수업을 들었는가의 여부가 머신러닝 수업의 학점과 상관관계가 있다고 말할 수 있다.
