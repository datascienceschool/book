Affinity Propagation#

Î™®Îì† Îç∞Ïù¥ÌÑ∞Í∞Ä ÌäπÏ†ïÌïú Í∏∞Ï§ÄÏóê Îî∞Îùº ÏûêÏã†ÏùÑ ÎåÄÌëúÌï† ÎåÄÌëú Îç∞Ïù¥ÌÑ∞Î•º ÏÑ†ÌÉùÌïúÎã§. ÎßåÏïΩ Ïä§Ïä§Î°úÍ∞Ä ÏûêÍ∏∞ ÏûêÏã†ÏùÑ ÎåÄÌëúÌïòÍ≤å ÎêòÎ©¥ ÌÅ¥Îü¨Ïä§ÌÑ∞Ïùò Ï§ëÏã¨Ïù¥ ÎêúÎã§.

  • responsibility \(r(i, k)\)

    • \(k\)Î≤àÏß∏ Îç∞Ïù¥ÌÑ∞Í∞Ä \(i\)Î≤àÏß∏ Îç∞Ïù¥ÌÑ∞Ïùò ÎåÄÌëúÍ∞Ä ÎêòÏñ¥Ïïº ÌïúÎã§Îäî Í∑ºÍ±∞

  • availability \(a(i, k)\)

    • \(i\)Î≤àÏß∏ Îç∞Ïù¥ÌÑ∞Í∞Ä \(k\)Î≤àÏß∏ Îç∞Ïù¥ÌÑ∞Î•º ÎåÄÌëúÎ°ú ÏÑ†ÌÉùÌï¥Ïïº ÌïúÎã§Îäî Í∑ºÍ±∞

  • responsibilityÏôÄ availabilityÎ•º Î™®Îëê 0ÏúºÎ°ú ÎÜìÍ≥† Îã§Ïùå ÏàòÏãùÏùÑ ÏàòÎ†¥Ìï† ÎïåÍπåÏßÄ Î∞òÎ≥µ

\[ r(i, k) \leftarrow s(i, k) - \max_{k' \neq k} ( a(i, k') + s(i, k')) \]
\[ a(i, k) \leftarrow \min(0, r(k, k) + \sum_{i' \neq i,k} r(i', k)) \]

Ïó¨Í∏∞ÏóêÏÑú \(s(i,k)\)Îäî Îã§ÏùåÍ≥º Í∞ôÏù¥ ÏùåÏùò Í±∞Î¶¨Î°ú Ï†ïÏùòÎêòÎäî Ïú†ÏÇ¨ÎèÑÏù¥Îã§.

\[ s(i,k) = -|| x_i - x_k ||^2 \]

ÌäπÌûà \(s(k,k)\)Îäî ÌäπÏ†ïÌïú ÏùåÏàò Í∞íÏúºÎ°ú ÏÇ¨Ïö©ÏûêÍ∞Ä Ï†ïÌï¥ Ï£ºÍ≤å ÎêòÎäîÎç∞ Ïù¥ Í∞íÏóê Îî∞ÎùºÏÑú ÌÅ¥Îü¨Ïä§ÌÑ∞Ïùò Í∞ØÏàòÍ∞Ä Îã¨ÎùºÏßÄÎäî ÌïòÏù¥Ìçº Î™®ÏàòÍ∞Ä ÎêúÎã§. \(s(k,k)\)Í∞Ä ÌÅ¨Î©¥ ÏûêÍ∏∞ ÏûêÏã†Ïóê ÎåÄÌïú Ïú†ÏÇ¨ÎèÑÍ∞Ä Ïª§Ï†∏ÏÑú ÌÅ¥Îü¨Ïä§ÌÑ∞Ïùò ÏàòÍ∞Ä Ï¶ùÍ∞ÄÌïúÎã§.

ÏúÑ ÏïåÍ≥†Î¶¨Ï¶òÏúºÎ°ú Í≥ÑÏÇ∞ÌïòÎäî \(r\), \(a\)Í∞Ä Îçî Ïù¥ÏÉÅ Î≥ÄÌôîÌïòÏßÄ ÏïäÍ≥† ÏàòÎ†¥ÌïòÎ©¥ Í≥ÑÏÇ∞Ïù¥ Ï¢ÖÎ£åÎêòÍ≥† Ï¢ÖÎ£å ÏãúÏ†êÏóêÏÑú \(r(k,k) + a(k,k) > 0\)Ïù¥ Îç∞Ïù¥ÌÑ∞Í∞Ä ÌÅ¥Îü¨Ïä§ÌÑ∞Ïùò Ï§ëÏã¨Ïù¥ ÎêúÎã§.

from sklearn.datasets.samples_generator import make_blobs
from sklearn.cluster import AffinityPropagation
from sklearn.metrics import *

centers = [[1, 1], [-1, -1], [1, -1]]
X, labels_true = make_blobs(n_samples=300, centers=centers, cluster_std=0.5, random_state=0)

model = AffinityPropagation(preference=-50).fit(X)

cluster_centers_indices = model.cluster_centers_indices_
labels = model.labels_
n_clusters_ = len(cluster_centers_indices)

print('Estimated number of clusters: %d' % n_clusters_)
print("Adjusted Rand Index: %0.3f" % adjusted_rand_score(labels_true, labels))
print("Adjusted Mutual Information: %0.3f" % adjusted_mutual_info_score(labels_true, labels))
print("Silhouette Coefficient: %0.3f" % silhouette_score(X, labels, metric='sqeuclidean'))
