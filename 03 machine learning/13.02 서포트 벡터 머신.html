\[ f(x^+) = w^Tx^+ - w_0 > 0 \]
\[ f(x^-) = w^Tx^- - w_0 < 0 \]

ÏÑúÌè¨Ìä∏Ïóê ÎåÄÌïú ÌåêÎ≥Ñ Ìï®ÏàòÏùò Í∞í \(f(x^+)\), \(f(x^-)\) Í∞íÏùÄ Î∂ÄÌò∏ Ï°∞Í±¥Îßå ÏßÄÌÇ§Î©¥ Ïñ¥Îñ§ Í∞íÏù¥ ÎêòÏñ¥ÎèÑ Í¥úÏ∞ÆÎã§. Îî∞ÎùºÏÑú Îã§ÏùåÍ≥º Í∞ôÏùÄ Ï°∞Í±¥ÏùÑ ÎßåÏ°±ÌïòÎèÑÎ°ù ÌåêÎ≥Ñ Ìï®ÏàòÎ•º Íµ¨ÌïúÎã§.

\[ f(x^+) = w^T x^{+} - w_0 = +1 \]
\[ f(x^-) = w^T x^{-} - w_0 = -1 \]

Í∑∏Î¶º 18.6.3 : ÏÑúÌè¨Ìä∏Î≤°ÌÑ∞Ïùò ÌåêÎ≥ÑÌï®Ïàò Í∞í

Ïù¥Î†áÍ≤å ÎêòÎ©¥ Î™®Îì† \(x_+\), \(x_+\) Îç∞Ïù¥ÌÑ∞Ïóê ÎåÄÌï¥ ÌåêÎ≥ÑÌï®ÏàòÏùò Í∞íÏùò Ï†àÎåÄÍ∞íÏù¥ 1Î≥¥Îã§ Ïª§ÏßÄÎØÄÎ°ú Îã§Ïùå Î∂ÄÎì±ÏãùÏù¥ ÏÑ±Î¶ΩÌïúÎã§.

\[ w^Tx_+ - w_o \geq 1 \]
\[ w^Tx_- - w_o \leq -1 \]

ÌåêÎ≥Ñ Í≤ΩÍ≥ÑÏÑ† \(w^T x - w_0=0\) Í≥º Ï†ê \(x^{+}\), \(x^{-}\) ÏÇ¨Ïù¥Ïùò Í±∞Î¶¨Îäî Îã§ÏùåÍ≥º Í∞ôÏù¥ Í≥ÑÏÇ∞Ìï† Ïàò ÏûàÎã§.

\[ \dfrac{w^T x^{+} - w_0}{\| w \|} = \dfrac{1}{\| w \|} \]
\[ -\dfrac{w^T x^{-} - w_0}{\| w \|} = \dfrac{1}{\| w \|} \]

Ïù¥ Í±∞Î¶¨Ïùò Ìï©ÏùÑ ÎßàÏßÑ(margin)Ïù¥ÎùºÍ≥† ÌïòÎ©∞ ÎßàÏßÑÍ∞íÏù¥ ÌÅ¥ ÏàòÎ°ù Îçî Í≤ΩÍ≥ÑÏÑ†Ïù¥ ÏïàÏ†ïÏ†ÅÏù¥ÎùºÍ≥† Î≥º Ïàò ÏûàÎã§. Í∑∏Îü∞Îç∞ ÏúÑÏóêÏÑú Ï†ïÌïú Ïä§ÏºÄÏùºÎßÅÏóê ÏùòÌï¥ ÎßàÏßÑÏùÄ Îã§ÏùåÍ≥º Í∞ôÏù¥ Ï†ïÎ¶¨ÎêúÎã§.

\[ \dfrac{w^T x^{+} - w_0}{\| w \|} -\dfrac{w^T x^{-} - w_0}{\| w \|} = \dfrac{2}{\| w \|}\]

ÎßàÏßÑ Í∞íÏù¥ ÏµúÎåÄÍ∞Ä ÎêòÎäî Í≤ΩÏö∞Îäî \(\| w \|\) Ï¶â, \(\| w \|^2\)Í∞Ä ÏµúÏÜåÍ∞Ä ÎêòÎäî Í≤ΩÏö∞ÏôÄ Í∞ôÎã§. Ï¶â Îã§ÏùåÍ≥º Í∞ôÏùÄ Î™©Ï†ÅÌï®ÏàòÎ•º ÏµúÏÜåÌôîÌïòÎ©¥ ÎêúÎã§.

\[ L = \dfrac{1}{2} ||w||^2 = \dfrac{1}{2} w^T w \]

ÎòêÌïú Î™®Îì† ÌëúÎ≥∏ Îç∞Ïù¥ÌÑ∞Ïóê ÎåÄÌï¥ Î∂ÑÎ•òÎäî Ï†úÎåÄÎ°ú ÎêòÏñ¥Ïïº ÌïòÎØÄÎ°ú Î™®Îì† Îç∞Ïù¥ÌÑ∞ \(x_i, y_i \; ( i = 1, \ldots, N)\)Ïóê ÎåÄÌï¥ Îã§Ïùå Ï°∞Í±¥ÏùÑ ÎßåÏ°±Ìï¥Ïïº ÌïúÎã§. ÏúÑÏóêÏÑú Ïä§ÏºÄÏùºÎßÅÏùÑ ÏÇ¨Ïö©ÌïòÏó¨ Î™®Îì† Îç∞Ïù¥ÌÑ∞Ïóê ÎåÄÌï¥ \(f(x_i) = w^Tx_i - w_o\) Í∞Ä 1Î≥¥Îã§ ÌÅ¨Í±∞ÎÇò -1Î≥¥Îã§ ÏûëÍ≤å ÎßåÎì§ÏóàÎã§Îäî Ï†êÏùÑ Ïù¥Ïö©ÌïúÎã§.

\[ y_i \cdot f(x_i) = y_i \cdot( w^Tx_i - w_o) \geq 1 \;\;\; ( i = 1, \ldots, N )\]
\[ y_i \cdot ( w^Tx_i - w_o) - 1 \geq 0 \;\;\; ( i = 1, \ldots, N )\]

ÎùºÍ∑∏ÎûëÏ£º ÏäπÏàòÎ≤ïÏùÑ ÏÇ¨Ïö©ÌïòÎ©¥ ÏµúÏÜåÌôî Î™©Ï†ÅÌï®ÏàòÎ•º Îã§ÏùåÍ≥º Í∞ôÏù¥ Í≥†ÏπòÎ©¥ ÎêúÎã§.

\[ L = \dfrac{1}{2} w^T w - \sum_{i=1}^N a_i \{ y_i \cdot ( w^Tx_i - w_o) - 1 \} \]

Ïó¨Í∏∞ÏóêÏÑú \(a_i\)ÏùÄ Í∞ÅÍ∞ÅÏùò Î∂ÄÎì±ÏãùÏóê ÎåÄÌïú ÎùºÍ∑∏ÎûëÏ£º ÏäπÏàòÏù¥Îã§.

Ïù¥ ÏµúÏ†ÅÌôî Î¨∏Ï†úÎ•º ÌíÄÏñ¥ \(w\), \(w_0\), \(a\)Î•º Íµ¨ÌïòÎ©¥ ÌåêÎ≥ÑÌï®ÏàòÎ•º ÏñªÏùÑ Ïàò ÏûàÎã§.

KKT(Karush‚ÄìKuhn‚ÄìTucker) Ï°∞Í±¥Ïóê Îî∞Î•¥Î©¥ Î∂ÄÎì±Ïãù Ï†úÌïú Ï°∞Í±¥Ïù¥ ÏûàÎäî Í≤ΩÏö∞ÏóêÎäî Îì±Ïãù Ï†úÌïúÏ°∞Í±¥ÏùÑ Í∞ÄÏßÄÎäî ÎùºÍ∑∏ÎûëÏ£º ÏäπÏàò Î∞©Î≤ïÍ≥º ÎπÑÏä∑ÌïòÏßÄÎßå \(i\)Î≤àÏß∏ Î∂ÄÎì±ÏãùÏù¥ ÏûàÏúºÎÇò ÏóÜÏúºÎÇò ÎãµÏù¥ Í∞ôÏùÄ Í≤ΩÏö∞ÏóêÎäî ÎùºÍ∑∏ÎûëÏßÄ ÏäπÏàòÏùò Í∞íÏù¥ \(a_i=0\)Ïù¥ ÎêúÎã§. Ïù¥ Í≤ΩÏö∞Îäî ÌåêÎ≥ÑÌï®ÏàòÏùò Í∞í \(w^Tx_i - w_o\)Ïù¥ \(-1\)Î≥¥Îã§ ÏûëÍ±∞ÎÇò \(1\)Î≥¥Îã§ ÌÅ∞ Í≤ΩÏö∞Ïù¥Îã§.

\[ y_i(w^Tx_i - w_o) - 1 > 0 \]

ÌïôÏäµ Îç∞Ïù¥ÌÑ∞ Ï§ëÏóêÏÑú ÏµúÏ†ÑÎ∞© Îç∞Ïù¥ÌÑ∞Ïù∏ ÏÑúÌè¨Ìä∏ Î≤°ÌÑ∞Í∞Ä ÏïÑÎãå Î™®Îì† Îç∞Ïù¥ÌÑ∞Îì§Ïóê ÎåÄÌï¥ÏÑúÎäî Ïù¥ Ï°∞Í±¥Ïù¥ ÎßåÏ°±ÎêòÎØÄÎ°ú ÏÑúÌè¨Ìä∏ Î≤°ÌÑ∞Í∞Ä ÏïÑÎãå Îç∞Ïù¥ÌÑ∞Îäî ÎùºÍ∑∏ÎûëÏßÄ ÏäπÏàòÍ∞Ä 0Ïù¥ÎùºÎäî Í≤ÉÏùÑ Ïïå Ïàò ÏûàÎã§.

\[ a_i = 0 \;\; \text{if} \;\; x_i \notin \{ x^{+}, x^{-} \} \]

ÎìÄÏñº ÌòïÏãù#

ÏµúÏ†ÅÌôî Ï°∞Í±¥ÏùÄ Î™©Ï†ÅÌï®Ïàò \(L\)ÏùÑ \(w\), \(w_0\)Î°ú ÎØ∏Î∂ÑÌïú Í∞íÏù¥ 0Ïù¥ ÎêòÏñ¥Ïïº ÌïòÎäî Í≤ÉÏù¥Îã§. $\( \dfrac{\partial L}{\partial w} = 0 \)$

\[ \dfrac{\partial L}{\partial w_0} = 0 \]

Ïù¥ ÏãùÏùÑ ÌíÄÏñ¥ÏÑú Ï†ïÎ¶¨ÌïòÎ©¥ Îã§ÏùåÍ≥º Í∞ôÏïÑÏßÑÎã§.

\[\begin{split} \begin{eqnarray} \dfrac{\partial L}{\partial w} &=& \dfrac{\partial}{\partial w} \left( \dfrac{1}{2} w^T w \right) - \dfrac{\partial}{\partial w} \sum_{i=1}^N \left( a_i y_i w^Tx_i - a_i y_i w_o - a_i \right) \\ &=& w - \sum_{i=1}^N a_i y_i x_i \\ &=& 0 \end{eqnarray} \end{split}\]
\[\begin{split} \begin{eqnarray} \dfrac{\partial L}{\partial w_0} &=& \dfrac{\partial}{\partial w_0} \left( \dfrac{1}{2} w^T w \right) - \dfrac{\partial}{\partial w_0} \sum_{i=1}^N \left( a_i y_i w^Tx_i - a_i y_i w_o - a_i \right) \\ &=& \sum_{i=1}^N a_i y_i \\ &=& 0 \end{eqnarray} \end{split}\]

Ï¶â,

\[ w = \sum_{i=1}^N a_i y_i x_i \]
\[ 0 = \sum_{i=1}^N a_i y_i \]

Ïù¥ Îëê ÏàòÏãùÏùÑ ÏõêÎûòÏùò Î™©Ï†ÅÌï®ÏàòÏóê ÎåÄÏûÖÌïòÏó¨ \(w\), \(w_0\)ÏùÑ ÏóÜÏï†Î©¥ Îã§ÏùåÍ≥º Í∞ôÎã§.

\[\begin{split} \begin{eqnarray} L &=& \dfrac{1}{2} w^T w - \sum_{i=1}^N a_i \{ y_i \cdot ( w^Tx_i - w_o) - 1 \} \\ &=& \dfrac{1}{2} \left( \sum_{i=1}^N a_i y_i x_i \right)^T \left( \sum_{j=1}^N a_j y_j x_j \right) - \sum_{i=1}^N a_i \left\{ y_i \cdot \left( \left( \sum_{j=1}^N a_j y_j x_j \right)^Tx_i - w_o \right) - 1 \right\} \\ &=& \dfrac{1}{2} \sum_{i=1}^N \sum_{j=1}^N a_i a_j y_i y_j x_i^T x_j - \sum_{i=1}^N \sum_{j=1}^N a_i a_j y_i y_j x_i^T x_j + w_0 \sum_{i=1}^N a_i y_i + \sum_{i=1}^N a_i \\ &=& \sum_{i=1}^N a_i - \dfrac{1}{2}\sum_{i=1}^N\sum_{j=1}^N a_i a_j y_i y_j x_i^T x_j \end{eqnarray} \end{split}\]

Ï¶â,

\[ L = \sum_{i=1}^N a_i - \dfrac{1}{2}\sum_{i=1}^N\sum_{j=1}^N a_i a_j y_i y_j x_i^T x_j \]

Ïù¥ Îïå \(a\)Îäî Îã§Ïùå Ï°∞Í±¥ÏùÑ ÎßåÏ°±ÌïúÎã§.

\[ \sum_{i=1}^N a_i y_i = 0 \]
\[ a_i \geq 0 \;\;\; ( i = 1, \ldots, N ) \]

Ïù¥ Î¨∏Ï†úÎäî \(w\)Î•º Íµ¨ÌïòÎäî Î¨∏Ï†úÍ∞Ä ÏïÑÎãàÎùº \(a\)ÎßåÏùÑ Íµ¨ÌïòÎäî Î¨∏Ï†úÎ°ú Î∞îÎÄåÏóàÏúºÎØÄÎ°ú ÎìÄÏñºÌòïÏãù(dual form)Ïù¥ÎùºÍ≥† ÌïúÎã§. ÎìÄÏñºÌòïÏãùÏúºÎ°ú Î∞îÍæ∏Î©¥ ÏàòÏπòÏ†ÅÏúºÎ°ú Î∞ïÏä§(Box)Ï†úÌïú Ï°∞Í±¥Ïù¥ ÏûàÎäî Ïù¥Ï∞®ÌîÑÎ°úÍ∑∏ÎûòÎ∞ç(QP; quadratic programming) Î¨∏Ï†úÍ∞Ä ÎêòÎØÄÎ°ú ÏõêÎûòÏùò Î¨∏Ï†úÎ≥¥Îã§Îäî Ìö®Ïú®Ï†ÅÏúºÎ°ú ÌíÄ Ïàò ÏûàÎã§.

ÎìÄÏñºÌòïÏãù Î¨∏Ï†úÎ•º ÌíÄÏñ¥ Ìï®Ïàò \(L\) Î•º ÏµúÏÜåÌôîÌïòÎäî \(a\)Î•º Íµ¨ÌïòÎ©¥ ÏòàÏ∏° Î™®ÌòïÏùÑ Îã§ÏùåÍ≥º Í∞ôÏù¥ Ïì∏ Ïàò ÏûàÎã§.

\[ f(x) = w^T x - w_0 = \sum_{i=1}^N a_i y_i x_i^T x - w_0 \]

\(w_0\)Îäî

\[ w_0 = w^T x^{+} - 1 \]

ÎòêÎäî
