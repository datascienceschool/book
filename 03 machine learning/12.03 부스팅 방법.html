ÏúÑ ÏãùÏóêÏÑú \(I\)Îäî \(k(x_i) \neq y_i\)ÎùºÎäî Ï°∞Í±¥Ïù¥ ÎßåÏ°±ÎêòÎ©¥ 1, ÏïÑÎãàÎ©¥ 0ÏùÑ Í∞ÄÏßÄÎäî ÏßÄÏãúÌï®Ïàò(indicator function)Ïù¥Îã§. Îî∞ÎùºÏÑú ÌãÄÎ¶∞ Î¨∏Ï†úÏóê ÎåÄÌïú Í∞ÄÏ§ëÏπòÏùò Ìï©Ïù¥Îã§.

ÏúÑÏõêÌöå \(C_m\)Ïóê Ìè¨Ìï®Îê† Í∞úÎ≥Ñ Î™®Ìòï \(k_m\)Ïù¥ ÏÑ†ÌÉùÎêú ÌõÑÏóêÎäî Í∞ÄÏ§ëÏπò \(\alpha_m\)Î•º Í≤∞Ï†ïÌï¥Ïïº ÌïúÎã§. Ïù¥ Í∞íÏùÄ Îã§ÏùåÏ≤òÎüº Í≥ÑÏÇ∞ÌïúÎã§.

\[ \epsilon_m = \dfrac{\sum_{i=1}^N w_{m,i} I\left(k_m(x_i) \neq y_i\right)}{\sum_{i=1}^N w_{m,i}} \]
\[ \alpha_m = \frac{1}{2}\log\left( \frac{1 - \epsilon_m}{\epsilon_m}\right) \]

Îç∞Ïù¥ÌÑ∞Ïóê ÎåÄÌïú Í∞ÄÏ§ëÏπò \(w_{m,i}\)Îäî ÏµúÏ¥àÏóêÎäî(m=1)Î™®Îì† Îç∞Ïù¥ÌÑ∞Ïóê ÎåÄÌï¥ Í∞ôÏùÄ Í∞íÏùÑ Í∞ÄÏßÄÏßÄÎßå ÏúÑÏõêÌöåÍ∞Ä Ï¶ùÍ∞ÄÌïòÎ©¥ÏÑú Í∞íÏù¥ Î∞îÎÄêÎã§. Í∞ÄÏ§ëÏπòÏùò Í∞íÏùÄ ÏßÄÏàòÌï®ÏàòÎ•º ÏÇ¨Ïö©ÌïòÏó¨ ÏúÑÏõêÌöå \(C_{m-1}\)Ïù¥ ÎßûÏ∂ò Î¨∏Ï†úÎäî ÏûëÍ≤å, ÌãÄÎ¶∞ Î¨∏Ï†úÎäî ÌÅ¨Í≤å ÌôïÎåÄ(boosting)ÎêúÎã§.

\[\begin{split} w_{m,i} = w_{m-1,i} \exp (-y_iC_{m-1}) = \begin{cases} w_{m-1,i}e^{-1} & \text{ if } C_{m-1} = y_i\\ w_{m-1,i}e & \text{ if } C_{m-1} \neq y_i \end{cases} \end{split}\]

\(m\)Î≤àÏß∏ Î©§Î≤ÑÏùò Î™®Îì† ÌõÑÎ≥¥Ïóê ÎåÄÌï¥ ÏúÑ ÏÜêÏã§ Ìï®ÏàòÎ•º Ï†ÅÏö©ÌïòÏó¨ Í∞ÄÏû• Í∞íÏù¥ ÏûëÏùÄ ÌõÑÎ≥¥Î•º \(m\)Î≤àÏß∏ Î©§Î≤ÑÎ°ú ÏÑ†Ï†ïÌïúÎã§.

ÏóêÏù¥Îã§Î∂ÄÏä§ÌåÖÏùÄ ÏÇ¨Ïã§ Îã§ÏùåÍ≥º Í∞ôÏùÄ ÏÜêÏã§Ìï®ÏàòÎ•º ÏµúÏÜåÌôîÌïòÎäî \(C_m\)ÏùÑ Ï∞æÏïÑÍ∞ÄÎäî Î∞©Î≤ïÏù¥ÎùºÎäî Í≤ÉÏùÑ Ï¶ùÎ™ÖÌï† Ïàò ÏûàÎã§.

\[ L_m = \sum_{i=1}^N \exp(‚àíy_i C_m(x_i)) \]

Í∞úÎ≥Ñ Î©§Î≤Ñ \(k_m\)Í≥º ÏúÑÏõêÌöåÏùò Í¥ÄÍ≥ÑÎäî

\[ C_m(x_i) = \sum_{j=1}^m \alpha_j k_j(x_i) = C_{m-1}(x_i) + \alpha_m k_m(x_i) \]

Ïù¥Í≥† Ïù¥ ÏãùÏùÑ ÎåÄÏûÖÌïòÎ©¥

\[\begin{split} \begin{eqnarray} L_m &=& \sum_{i=1}^N \exp(‚àíy_i C_m(x_i)) \\ &=& \sum_{i=1}^N \exp\left(‚àíy_iC_{m-1}(x_i) - \alpha_m y_i k_m(x_i) \right) \\ &=& \sum_{i=1}^N \exp(‚àíy_iC_{m-1}(x_i)) \exp\left(-\alpha_m y_i k_m(x_i)\right) \\ &=& \sum_{i=1}^N w_{m,i} \exp\left(-\alpha_m y_i k_m(x_i)\right) \\ \end{eqnarray} \end{split}\]

\(y_i\)ÏôÄ \(k_M(x_i)\)Í∞Ä 1 ÎòêÎäî -1Í∞íÎßå Í∞ÄÏßà Ïàò ÏûàÎã§Îäî Ï†êÏùÑ Ïù¥Ïö©ÌïòÎ©¥,

\[\begin{split} \begin{eqnarray} L_m &=& e^{-\alpha_m}\sum_{k_m(x_i) = y_i} w_{m,i} + e^{\alpha_m}\sum_{k_m(x_i) \neq y_i} w_{m,i} \\ &=& \left(e^{\alpha_m}-e^{-\alpha_m}\right) \sum_{i=1}^N w_{m,i} I\left(k_m(x_i) \neq y_i\right) + e^{-\alpha_m}\sum_{i=1}^N w_{m,i} \end{eqnarray} \end{split}\]

\(L_m\)ÏùÑ ÏµúÏÜåÌôîÌïòÎ†§Î©¥ \(\sum_{i=1}^N w_{m,i} I\left(k_m(x_i) \neq y_i\right)\)ÏùÑ ÏµúÏÜåÌôîÌïòÎäî \(k_m\) Ìï®ÏàòÎ•º Ï∞æÏùÄ Îã§Ïùå \(L_m\)ÏùÑ ÏµúÏÜåÌôîÌïòÎäî \(\alpha_m\)ÏùÑ Ï∞æÏïÑÏïº ÌïúÎã§.

\[ \dfrac{d L_m}{d \alpha_m} = 0 \]

Ïù¥ Ï°∞Í±¥ÏúºÎ°úÎ∂ÄÌÑ∞ \(\alpha_m\) Í≥µÏãùÏùÑ Ïú†ÎèÑÌï† Ïàò ÏûàÎã§.

class MyAdaBoostClassifier(AdaBoostClassifier):
    
    def __init__(self,
                 base_estimator=None,
                 n_estimators=50,
                 learning_rate=1.,
                 algorithm='SAMME.R',
                 random_state=None):

        super(MyAdaBoostClassifier, self).__init__(
            base_estimator=base_estimator,
            n_estimators=n_estimators,
            learning_rate=learning_rate,
            random_state=random_state)
        self.sample_weight = [None] * n_estimators
        
    def _boost(self, iboost, X, y, sample_weight, random_state):
        sample_weight, estimator_weight, estimator_error = \
        super(MyAdaBoostClassifier, self)._boost(iboost, X, y, sample_weight, random_state)
        self.sample_weight[iboost] = sample_weight.copy()
        return sample_weight, estimator_weight, estimator_error

\[ x_{m} = x_{m-1} - \alpha_m \dfrac{df}{dx} \]

Í∑∏Î†àÎîîÏñ∏Ìä∏ Î∂ÄÏä§Ìä∏ Î™®ÌòïÏóêÏÑúÎäî ÏÜêÏã§ Î≤îÌï®Ïàò(loss functional) \(L(y, C_{m-1})\)ÏùÑ ÏµúÏÜåÌôîÌïòÎäî Í∞úÎ≥Ñ Î∂ÑÎ•òÌï®Ïàò \(k_m\)Î•º Ï∞æÎäîÎã§. Ïù¥Î°†Ï†ÅÏúºÎ°ú Í∞ÄÏû• ÏµúÏ†ÅÏùò Ìï®ÏàòÎäî Î≤îÌï®ÏàòÏùò ÎØ∏Î∂ÑÏù¥Îã§.

\[ C_{m} = C_{m-1} - \alpha_m \dfrac{\delta L(y, C_{m-1})}{\delta C_{m-1}} = C_{m-1} + \alpha_m k_m \]

Îî∞ÎùºÏÑú Í∑∏Î†àÎîîÏñ∏Ìä∏ Î∂ÄÏä§Ìä∏ Î™®ÌòïÏùÄ Î∂ÑÎ•ò/ÌöåÍ∑Ä Î¨∏Ï†úÏóê ÏÉÅÍ¥ÄÏóÜÏù¥ Í∞úÎ≥Ñ Î©§Î≤Ñ Î™®ÌòïÏúºÎ°ú ÌöåÍ∑ÄÎ∂ÑÏÑù Î™®ÌòïÏùÑ ÏÇ¨Ïö©ÌïúÎã§. Í∞ÄÏû• ÎßéÏù¥ ÏÇ¨Ïö©ÎêòÎäî ÌöåÍ∑ÄÎ∂ÑÏÑù Î™®ÌòïÏùÄ ÏùòÏÇ¨Í≤∞Ï†ï ÌöåÍ∑ÄÎÇòÎ¨¥(decision tree regression model) Î™®ÌòïÏù¥Îã§.

Í∑∏Î†àÎîîÏñ∏Ìä∏ Î∂ÄÏä§Ìä∏ Î™®ÌòïÏóêÏÑúÎäî Îã§ÏùåÍ≥º Í∞ôÏùÄ Í≥ºÏ†ïÏùÑ Î∞òÎ≥µÌïòÏó¨ Î©§Î≤ÑÏôÄ Í∑∏ Í∞ÄÏ§ëÏπòÎ•º Í≥ÑÏÇ∞ÌïúÎã§.

  1. \(-\tfrac{\delta L(y, C_m)}{\delta C_m}\) Î•º Î™©ÌëúÍ∞íÏúºÎ°ú Í∞úÎ≥Ñ Î©§Î≤Ñ Î™®Ìòï \(k_m\) ÏùÑ Ï∞æÎäîÎã§.
